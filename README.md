# Movie Website

A Django-based web application that showcases movies. Users can view movie posters, click on them, and see detailed information including where to watch each movie.

## Features

- Browse a list of movies with posters
- Click on a poster to view movie details
- See streaming platforms or where to watch
- Built with Django and Python
